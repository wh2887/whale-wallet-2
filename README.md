# 鲸鱼记账
一款基于 Vue 技术栈开发的简介本地记账WebApp

![](https://gitee.com/hellow2887/blogimage/raw/master/img/qrCode.png)

## 技术栈
- Vue
- Vuex
- VueRouter
- TypeScript
- SCSS

## 功能
- 记录账单（包括金额、日期、分类等）
- 分类编辑（增、改）
- 账单明细
- 近 30 天图表展示


## 安装依赖
```
yarn install
```

### 开发
```
yarn serve
```

### 构建
```
yarn build
```
